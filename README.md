# 🚨 Women Safety System

## 📌 Description
The Women Safety System is a web-based application designed to enhance women's safety by providing emergency assistance features. It allows users to send alerts, share location, and quickly access help during critical situations.

---

## 🚀 Features
- 🚨 Emergency SOS alert system  
- 📍 Real-time location sharing  
- 📱 User-friendly interface  
- 🔐 Secure user authentication  
- 📞 Quick access to emergency contacts  
- 🧑‍💼 Admin monitoring system  

---

## 🛠️ Tech Stack

### 💻 Frontend
- HTML
- CSS
- JavaScript

### ⚙️ Backend
- Node.js
- Express.js

### 🗄️ Database
- MongoDB

---

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/codewithshivaa/women-safety-system.git
cd women-safety-system


2. Install dependencies

npm install


3. Run the project

npm start


---

## 🔐 Environment Variables

Create a `.env` file:


MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
